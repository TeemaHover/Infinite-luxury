## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker

```bash
# .ENV LOCAL
$ DB_HOST=localhost
$ DB_PORT=5436
$ DB_NAME=postgres
$ DB_POOL_SIZE=5
$ DB_USER=postgres
$ DB_PASSWORD=postgres
```
